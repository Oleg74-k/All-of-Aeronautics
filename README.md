# All Of Aeronautics

Кастомная технологическая сборка модов на базе NeoForge (1.21.1).

---

## Установка клиента (Prism Launcher)

1. Скачайте архив клиента `All_Of_Aeronautics_Client.zip` из раздела **Releases**.
2. Откройте **Prism Launcher**.
3. Нажмите кнопку **Добавить сборку** (Add Instance) на верхней панели.
4. Выберите вкладку **Импорт** (Import) слева.
5. Нажмите **Обзор** (Browse) и выберите скачанный архив `All_Of_Aeronautics_Client.zip`.
6. Нажмите **ОК**. Лаунчер сам распакует сборку и скачает необходимые библиотеки и ванильные файлы.
7. Запустите сборку.

---

## Запуск сервера

1. Скачайте архив сервера `All_Of_Aeronautics_Server.zip` из раздела **Releases** и распакуйте его в пустую папку.
2. Убедитесь, что у вас установлена **Java 21**.
3. Запустите файл **`run.bat`** (Windows) или **`run.sh`** (Linux/macOS) для старта сервера.
   * *Примечание:* Для настройки выделения ОЗУ отредактируйте файл `user_jvm_args.txt`.
